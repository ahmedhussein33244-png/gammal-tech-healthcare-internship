Healthcare Reviews Analysis

Overview
This dataset contains 50 negative reviews collected from healthcare apps in Egypt.

Apps Covered
- Vezeeta
- Sehaty
- Altibbi
- Yodawy
- Chefaa

Categories
- Performance
- Trust Issue
- Pricing
- Bug
- Bad UX
- Missing Feature
- Support Issue
- Logistics
- Payment Issue
- Ads

Purpose
To identify common user pain points and improve healthcare app experience.
